next js set up getting started
==================================================

i started a new next js project. here are the steps i followed to set it up.

1. installed node.js and npm on my machine.
2. created a new next js app using the command:
   ```
   npx create-next-app@latest nextjs
   ```

3. started the development server:
   ```
   cd nextjs
   npm run dev
   ```
4. opened my browser and navigated to `http://localhost:3000` to see the default next js welcome page.
